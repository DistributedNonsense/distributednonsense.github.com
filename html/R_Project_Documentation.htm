The RiakClient.Comms.LoadBalancing namespace provides classes that are related to load balancing between Riak nodes.

The RiakClient.Config namespace provides classes for interacting with XML configuration files.

The RiakClient.Exceptions namespace contains some useful extension methods for interacting with the client.

The RiakClient.Models namespace contains all the models used within the library.
