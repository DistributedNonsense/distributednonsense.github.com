Get the specified key from the bucket. Optionally can be read from rVal instances. By default, the server's r-value will be used, but can be overridden by rVal.

Remarks
If a node does not respond, that does not necessarily mean that the bucket/key combination is not available. It simply means that fewer than R/PR nodes responded to the read request. See RiakGetOptions for information on how different options change Riak's default behavior.
